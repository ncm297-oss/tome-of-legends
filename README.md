# Tome of Legends

D&D 5e Character Manager built with React + Vite + Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push to a GitHub repo
2. Go to [vercel.com](https://vercel.com), sign in with GitHub
3. Click "Add New Project", select the repo
4. Leave all settings default, click Deploy
